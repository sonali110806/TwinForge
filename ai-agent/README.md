# AI Agent Module 🚀

This module is part of the TwinForge project.

## 🔧 Features
- Detect → Decide → Act loop
- Anomaly detection
- Root cause analysis (AI-based prompt)
- Fix tournament system
- Confidence-based decision making
- Post-mortem report generation

## 📁 Files
- main.py → Main execution loop
- detector.py → Detect anomalies
- decision.py → Root cause & fixes
- tournament.py → Select best fix
- confidence.py → Decision logic
- agent.py → AI prompt generation
- postmortem.py → Report generation

## 🧠 Working
The system continuously monitors metrics, detects anomalies, analyzes root cause, tests multiple fixes, and applies the best solution based on confidence.
